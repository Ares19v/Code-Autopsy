# Security Policy

## Supported Versions

Currently, only the main branch is receiving security updates.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | :white_check_mark: |
| others  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Code Autopsy, please send an e-mail to the repository owner or open an issue on GitHub. We will address all security vulnerabilities promptly.

Please do not disclose the vulnerability publicly until a fix has been released.
