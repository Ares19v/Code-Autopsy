This project and everyone participating in it is governed by the [Code Autopsy Code of Conduct](CODE_OF_CONDUCT.md). By participating, you are expected to uphold this code.

## I Have a Question

Before you ask a question, it is best to search for existing [Issues](https://github.com/Ares19v/Code-Autopsy/issues) that might help you. In case you have found a suitable issue and still need clarification, you can write your question in this issue.

## I Want To Contribute

### Reporting Bugs

Bugs are tracked as [GitHub issues](https://github.com/Ares19v/Code-Autopsy/issues). Explain the problem and include additional details to help maintainers reproduce the problem:

* **Use a clear and descriptive title** for the issue to identify the problem.
* **Describe the exact steps which reproduce the problem** in as many details as possible.
* **Provide specific examples to demonstrate the steps.** Include links to files or GitHub projects, or copy/pasteable snippets.

### Suggesting Enhancements

Enhancement suggestions are tracked as [GitHub issues](https://github.com/Ares19v/Code-Autopsy/issues).

* **Use a clear and descriptive title** for the issue to identify the suggestion.
* **Provide a step-by-step description of the suggested enhancement** in as many details as possible.
* **Explain why this enhancement would be useful** to most Code Autopsy users.

### Pull Requests

* Fill in the required template
* Do not include issue numbers in the PR title
* Include screenshots and animated GIFs in your pull request whenever possible
* End files with a newline
* Make sure the CI (linting and tests) passes before asking for a review
